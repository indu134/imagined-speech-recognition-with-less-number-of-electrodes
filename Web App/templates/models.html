Model Description

Description

This model utilizes a neural network architecture comprising two LSTM layers followed by fully connected layers. The first LSTM layer has 128 units and the second has 64 units, capturing sequential dependencies in the input data. The LSTM outputs are then flattened into a one-dimensional vector. Two Dense layers follow, with 32 units and 3 softmax activations for 3-word classification and 2 softmax activations for 2-word classification, forming the final prediction layer.
